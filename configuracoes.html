Como publicar no GitHub Pages
1
Crie uma conta em github.com se ainda não tiver
2
Crie um repositório novo chamado atelie (público)
3
Faça upload de todos os arquivos do sistema
4
Vá em Settings → Pages → Source: Deploy from branch → main → / (root) → Save
5
Seu sistema estará em seuusuario.github.io/atelie
Como configurar o Google OAuth
1
Acesse console.cloud.google.com com a conta do Google
2
Crie um novo projeto chamado "Ateliê da Costura"
3
Vá em APIs e Serviços → Ativar: Google Sheets API e Google Drive API
4
Vá em Credenciais → Criar credenciais → ID do cliente OAuth 2.0 → Aplicativo da Web
5
Em "Origens JavaScript autorizadas", adicione sua URL do GitHub Pages (ex: https://seuusuario.github.io)
6
Copie o Client ID gerado e cole no arquivo js/config.js no campo CLIENT_ID
7
Crie uma planilha Google Sheets em branco, copie o ID da URL (entre /d/ e /edit) e cole no campo SHEET_ID do mesmo arquivo
